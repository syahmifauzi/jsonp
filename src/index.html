Send a POST request to https://json-p.herokuapp.com endpoint with the following JSON body 

      {
        "inArguments": [
          {
            "message": "Hello telcowin this is a test",
            "from": "01234567890",
            "to": "0199999999"
          }
        ],
        "activityObjectID": "7d3a5a3e-9717-40fe-b2f1-e72270e3e950",
        "journeyId": "18297097-9f4f-4eb1-810e-8507ad4ecc6b",
        "activityId": "7d3a5a3e-9717-40fe-b2f1-e72270e3e950",
        "definitionInstanceId": "d26ad8fc-b9c3-4f4b-b709-bce185d9cd14",
        "activityInstanceId": "512277ab-9a44-4fcf-883f-473e52c7fb98",
        "keyValue": "telcowin@telcowin.com",
        "mode": 0
      }

The response will be anything in "inArguments" first object. In our case, it will be: 

      {
        "message": "Hello telcowin this is a test",
        "from": "01234567890",
        "to": "0199999999"
      }

Before sending the response, the server will send the above data to https://eoitnqeyosaiq64.m.pipedream.net endpoint.
It will then update this Google Sheet with the data.
